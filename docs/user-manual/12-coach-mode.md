# Coach Mode

## What is this

Coach Mode transforms technical numbers into plain, actionable language suited for the locker room and the bench. It's designed for moments when you need to communicate quickly — halftime huddles, timeouts, pre-game talks — without stopping to explain statistics.

## How to activate it

Click the **Coach Mode** button in the top-right header bar. It turns amber/orange when active.

![Coach Mode toggle](./screenshots/09-game-day-coach-mode.png)

Your preference is saved automatically and persists across page navigation and browser sessions.

## What changes when Coach Mode is ON

| Feature | Normal Mode | Coach Mode |
|---------|-------------|------------|
| Win Probability | "62% win probability" | "6 wins in 10" |
| Halftime Re-sim | Shows % + statistical rationale | Short action commands only |
| Keys to Victory descriptions | Full analytical text | Descriptions hidden |
| Priority Keys panel | Shows full description | Title only |
| Matchup Workspace | Full win % number | "X in 10" format |

## When to use it

- **Pre-game**: Project the workspace on a screen for players — coach mode hides confusing stats
- **Halftime**: Run halftime re-sim and read the adjustments aloud directly from the screen
- **Timeout**: Glance at the Game Plan Alignment panel without needing to interpret z-scores

## Tips
- You can toggle Coach Mode on and off as often as needed; it never changes the underlying data
- Use it during team meetings; switch it off when doing your own analysis

## Common pitfalls
- "Coach Mode toggle doesn't appear" → Make sure you're on a page with a header (any page except Login)
- "Numbers still show after enabling" → Refresh the page if the context didn't propagate
