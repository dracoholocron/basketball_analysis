# Scenario: Live Game Night

**Audience**: Coach or assistant with a tablet/laptop courtside  
**Time required**: Full game (live)  
**Goal**: Track every play, stay aligned with game plan, and get AI halftime adjustments

---

## Pre-Game (30 min before tip-off)

1. Open **Game Tracker** → select tonight's matchup
2. Verify the **Game Plan Alignment** panel on the right shows your Top 3 Priority Keys
3. Check the clock settings: sport, periods, minutes per period, timeouts per team
4. Enable **Coach Mode** if you'll be showing the screen to players

---

## First Half: Logging Events

1. Tap anywhere on the court diagram to place a shot event
2. Select the team (T1 = Your Team / T2 = Opponent) and the event type
3. After a missed shot, the **Follow-Up modal** appears — quickly log the rebound and jersey number
4. Watch the **Game Plan Alignment panel**:
   - 🟢 **REINFORCED** = Keep going, it's working
   - 🔴 **DRIFTING** = This key is slipping, communicate an adjustment
   - ⚪ **IN_FLOW** = Neutral / not enough data

5. Use the timeout buttons to track timeouts used by each team

---

## Halftime

1. Click **Halftime Re-sim** in the toolbar
2. Wait ~5 seconds → you'll see:
   - Updated win probability based on first-half data
   - 2-3 **halftime adjustments** with HIGH/MEDIUM/LOW priority
3. In Coach Mode, adjustments appear as direct commands (e.g. "PUSH THE PACE")
4. Read the HIGH priority adjustment aloud in the locker room

---

## Second Half

5. Continue logging events as in the first half
6. The alignment panel now uses re-sim data for more accurate status
7. Use **Undo** if you mislog an event

---

## Post-Game

8. Click **Event Heatmap** to review:
   - Shot chart with court zones
   - FG%, blocks, steals, fouls summary
   - Key-by-key progress bars showing how each priority performed
9. Go to **Games → Upload Video** if you have game film
10. Once video analysis is complete, view the **tracking overlay** on the game detail page

---

## Tips
- Assign one assistant coach specifically to log events so you can focus on coaching
- You can open Game Tracker on a phone/tablet while watching the game remotely via stream
